# Client - ReactJS Frontend

This directory will contain the ReactJS frontend microservice for the authentication service.

## Structure

```
client/
├── public/          # Static files
├── src/            # Source code
│   ├── components/ # React components
│   ├── services/   # API services
│   ├── utils/      # Utility functions
│   └── App.js      # Main application
├── package.json    # Dependencies
└── README.md       # This file
```

## Getting Started

(To be added when implementing the ReactJS frontend)

### Prerequisites

- Node.js >= 16.x


```bash

```

```bash
npm run build
```
